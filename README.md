# Видеоплеер

Сайт с видеоплеером.  

## Интерфейс видеоплеера

Чтобы ознакомиться с интерфейсом видеоплеера, перейдите по ссылке: [Смотреть](https://linalleks.github.io/Django-layout-L2/)  
Вы увидите страницу, как на скриншоте:

![Скриншот сайта](screenshots/Screenshot_1.png)

Видеоплеер оснащен панелью управления. На следующем скриншоте показаны 2 варианта панели, чтобы было видно все инструменты управления, а ниже их описания:  
![Запуск](screenshots/Screenshot_2.png)

1. Кнопка запуска видео-плеера. После нажатия (запуска видео) автоматически заменяется на кнопку паузы и появляется снова после окончания видео. 
2. Кнопка паузы видео-плеера. После нажатия (остановки видео) автоматически заменяется на кнопку запуска.
3. Кнопка, которая отключает звук плеера. После нажатия (отключения звука) автоматически заменяется на кнопку включения звука.
4. Кнопка, которая включает звук. После нажатия (включения звука) автоматически заменяется на кнопку отлючения звука.
5. Текущее время видео.
6. Общая продолжительность видео.
7. Закрашенная линия прокрутки видео, показывает какая часть видео была просмотрена.
8. Ползунок для прокрутки видео, позволяет быстро промотать на интересный момент ролика. Управляется с помощью зажатой левой кнопки мыши. Отображается только при наведении курсора мыши на панель управления.
9. Кнопка, которая включает полноэкранный режим. Выйти из данного режима можно с помощью клавиши `Esc` на клавиатуре.
